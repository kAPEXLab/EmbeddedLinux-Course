Directory Navigation and File Operations  
• pwd # Show current directory  
• cd /path/to/dir # Change directory  
• ls -l # List files with details  
• mkdir new_folder # Create a directory  
• rm filename.txt # Remove a file  
• rmdir old_folder # Remove an empty directory  
• cp file1 file2 # Copy file  
• mv file1 newname # Rename or move file  
Example  
$ mkdir project  
# create directory - project  
# go to project directory  
$ cd project  
$ touch main.c  
# creae main.c file  
$ ls –l  
# list all the files with details  
# copy main → backup_main  
# move backup_main → old_main  
$ cp main.c backup_main.c  
$ mv backup_main.c old_main.c

Permissions and Access Control  
• ls –l  
# list the files with all permissions  
• chmod 660 file  
# change file permissions, in order: user group others, 6 = rw-, 0 = ---  
• chown user:group /dev/mydevice

Process and Thread Management  
• ps -ef | grep lane_detect → find running process  
• pidof ./a.out → get PID directly  
• strace -f -e trace=process ./lane_detect → trace forks/execs  
• kill -9 <pid> → forceful termination  
• nice -n 10 ./a.out  
• renice -n -5 <pid>  
• pstree -p <pid>  
Task 7 - Identify Processes & Trace activity  
1. Start a sample program or sleep process  
sleep 500 &  
2. Find its PID using:  
• ps -ef | grep sleep  
• pidof sleep  
3. Trace Its Activity & observe which all system calls are logged  
strace -f -e trace=process sleep 1
