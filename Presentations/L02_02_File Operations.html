Why File Operations Matter?  
• Everything in Linux is a file  
• Regular files  
• Directories  
• Devices (/dev/ttyS0, /dev/i2c-1)  
• Sockets  
• Pipes  
• System programming starts with file operations  
• Low-level, unbuffered, deterministic I/O  
• Sensors, UART, GPIO, sysfs entries, I2C/SPI, flash devices ― all exposed as files.

open()  
• The open() system call opens the file specified by path.  
• Headers Required  
#include <fcntl.h>  
int open(const char *path, int flags, int mode );  
int open(const char *path, int flags);  
• path: file path  
• Flags:  
• O_RDONLY, O_WRONLY, O_RDWR  
• O_CREAT (create if not exists; requires a mode like 0644)  
• O_TRUNC (truncate to 0 on open if writing)  
• O_APPEND (writes always append to end)  
• O_EXCL (with O_CREAT, fail if file exists)  
• mode: file perms used only when O_CREAT  
• Return: new file descriptor >=0. On error: -1  
#include <fcntl.h>  
#include <errno.h>  
#include <stdio.h>

read()  
• Used to read the data from file  
• Headers Required  
#include <unistd.h>  
ssize_t read(int fd, void *buffer, size_t count)  
• fd: file descriptor of the file to be read  
• Buffer – provides the address of memory buffer where data needs to be placed  
• count: number of bytes to read  
• Return number of bytes read, -1 on error  
Sample Code – Read from file  
char buf[BUF_SIZE];  
ssize_t nread;  
nread = read(fd, buf, 20);  
if (nread == -1) {  
printf("read() failed %d\n", errno);  
perror("ERROR");  
return 1;  
}
else{  
buf[nread] = '\0’;  
printf("The number of bytes read is %ld and read data is \"%s\" \n", nread, buf);  
}
write()  
• Used to write the data to the opened file  
• #include <unistd.h>  
ssize_t write(int fd, void *buffer, size_t count)  
• Return number of bytes written  
• Buffer – provides the address of memory buffer where data needs to be placed  
Sample Code – Write to file  
const char *msg = "Hello from demoNew.txt!\nThis line was written by write().\n";  
ssize_t to_write = (ssize_t)strlen(msg);  
ssize_t written = write(fd, msg, (size_t)to_write);  
if (written == -1) {  
printf("write() failed %d\n", errno);  
perror("ERROR");  
return 1;  
}
else{  
printf("The number of bytes written is %ld \n", written);  
}

close()  
• Used to close a file.  
• close() reduces refcount of open file description.  
• After close(), FD can be reused by open().  
int close(int fd)  
• fd: file descriptor to close.  
if (close(fd) == -1){  
printf("close() failed %d\n", errno);  
}
else{  
printf("The file is closed successfully!\n");  
}
